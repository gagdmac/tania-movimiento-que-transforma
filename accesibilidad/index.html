Nuestro compromiso

Tania Angulo se compromete a garantizar la accesibilidad digital para personas con diversas capacidades. Trabajamos continuamente para mejorar la experiencia de usuario de todos nuestros visitantes y aplicar los estándares de accesibilidad pertinentes.

Estándar de conformidad

Este sitio web aspira a cumplir con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.2 de nivel AA del W3C. Estas pautas explican cómo hacer el contenido web más accesible para personas con discapacidades y más amigable para todos los usuarios.

Medidas implementadas

HTML semántico con estructura correcta de encabezados y regiones landmark.

Textos alternativos descriptivos en todas las imágenes.

Navegación completa por teclado con indicadores de foco visibles.

Contraste de color que cumple los ratios mínimos de WCAG 2.2 AA.

Panel de opciones de accesibilidad con tamaño de texto, alto contraste, modo oscuro, reducción de movimiento, resaltado de foco y subrayado de enlaces.

Soporte multilingüe (español, inglés, portugués).

Diseño responsivo que se adapta a diferentes dispositivos y tamaños de pantalla.

Enlace para saltar al contenido principal.
